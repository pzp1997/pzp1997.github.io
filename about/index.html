About

Palmer Paul

Hey there! Welcome to my site! I'm Palmer, and I am a self-taught programmer. I am very proficient in many languages, and I guess you could say that I have a knack for programming. At the moment, I am just trying to broaden my horizons as much as possible by doing various interesting projects, which can be found on my GitHub.

My programming career started in 2012 when my school's Engineering Club was building a rudimentary 3D printer, and we needed someone to design some software for it. To suit up for the task, I started learning Python. After completing that project I knew I had caught the coding bug, and went on to teach myself JavaScript, Processing, Java (including the whole AP Computer Science curriculum), and a slew of other programming languages.

Last year I created a Content Management System (CMS) with a friend of mine in Computer Programming Club. I wrote the entire backend using Node.js (Express and MongoDB), as well as a good portion of the frontend using Jade (a nifty little HTML templating language), CSS, and JavaScript with jQuery.

Over the summer, I experimented with computer vision using the OpenCV library. My primary project was finding an effective method of determining the gender of a person in real-time using full body images (i.e. no face detection). This project is still a work in progress.

Recently, I have experimented with Johnny Five (a JavaScript robotics programming framework), written a geolocation extension for ScratchX, and started learning C. In addition to all of the above, I am brushing up on my Java knowledge in preparation for the AP Computer Science exam in May.
